# Афанасово — концепт-генплан коттеджного посёлка

**🌐 Живой веб-альбом:** https://ismailovvism-oss.github.io/afonasovo/ · **🚶 3D-прогулка:** https://ismailovvism-oss.github.io/afonasovo/walk.html

Пилотный проект градостроительной документации концепт-стадии (ПП) для посёлка
в н.п. Большое Афанасово (Нижнекамский р-н, Республика Татарстан). Весь комплект
строится **автоматически из топосъёмки заказчика**: DWG → редактируемый DXF
(МСК-16) → альбом листов (PDF) → ТЭП → 3D-визуализация.

## Ключевые числа

- Участок **17,69 га** (176 895 м²), перепад рельефа ~21 м (578 отметок).
- **158 участков** = 131 одноквартирный (10×10) + 17 двухквартирных (10×18)
  + 10 таунхаусов (10×28) → **205 жилых единиц**.
- Соцобъекты: школа-детсад, мечеть, магазин, спортивная и детская площадки.
- ТЭП сведён к 100%.

## Как воспроизвести

```bash
bash pipeline/setup.sh          # Python-окружение + LibreDWG (dwg2dxf)
pipeline/venv/bin/python pipeline/generate_genplan.py \
    техзадание/*.dwg --out результат_нового_проекта \
    --boundary-color 1 --target-ha 18
```

Подробности и параметры — в [`pipeline/README.md`](pipeline/README.md).

## Технологии

`LibreDWG` (dwg2dxf) → `ezdxf` (чтение геометрии) → `shapely` (нарезка кварталов,
посадка домов, площади) → `matplotlib` (листы/рендеры) → `ffmpeg` (облёт) →
`three.js` (3D-прогулка). Редактируемый выход — DXF в МСК-16 для nanoCAD / AutoCAD.

## Статус и границы применимости

Это **концепт-стадия (ПП)**, стартовая заготовка для доводки человеком в CAD, а не
рабочая документация под стройку. Регулярная сетка не оптимизирована под рельеф;
трассировку улиц по горизонталям, детальную раскладку сетей, планы этажей и фасады
зданий доделывает проектировщик. Подробнее — в разделе «Границы применимости»
файлов README подпапок.
