# Гайд: из `village.obj` → фотореалистичный кадр и HD-облёт в Blender

Для новичка на Linux. Всё бесплатно. Время: первый красивый кадр — ~20 минут.

---

## 0. Установка Blender (Linux)

```bash
flatpak install flathub org.blender.Blender
flatpak run org.blender.Blender
```
(или скачай с blender.org, распакуй, запусти `./blender`). Версия 4.x.

Интерфейс: вращать вид — **колесо мыши зажать**, приблизить — **скролл**,
двигать — **Shift+колесо**. Слева меню, справа сцена, снизу таймлайн.

---

## 1. Импорт модели

`File → Import → Wavefront (.obj)`. В окне импорта **справа important**:
- **Up Axis: Z**, **Forward: Y** (у нас высота — это Z; иначе дом ляжет набок).
- Выбери `village.obj` → Import.

Появятся два объекта: `terrain` и `buildings` (с материалами из `village.mtl`).

> Если модель огромная/крошечная — она в **метрах** (реальный масштаб). Нажми `N`
> → вкладка Item → увидишь размеры. Чтобы вписать в кадр: наведи мышь на сцену,
> нажми `Home` (показать всё).

---

## 2. Починить нормали (важно!)

Наши грани могут смотреть «внутрь» — из-за этого поверхности темнеют.
1. Выдели объект `buildings` (клик), **Tab** → режим редактирования.
2. **A** (выделить всё) → **Shift+N** (пересчитать нормали наружу).
3. **Tab** → назад. Повтори для `terrain`.

---

## 3. Свет и небо (за 1 минуту)

1. `Add → Light → Sun`. В свойствах лампы (значок лампочки справа) поставь
   **Strength ≈ 3**, **Angle ≈ 3°** (мягкие тени). Поверни солнце: `R` → мышь →
   клик, чтобы свет падал сбоку (даёт объём).
2. Небо-подсветка: справа вкладка **World** (глобус) → Color кликни →
   выбери светло-голубой, Strength ≈ 0.4. Это имитация неба.

---

## 4. Материалы (делают всю «сочность»)

Справа вкладка **Material** (шарик). Выдели `terrain`:
- Base Color → зелёный (трава), **Roughness ≈ 0.9** (не блестит).
Выдели `buildings`:
- Base Color → бежевый (стены). Roughness ≈ 0.7.

**Крыши отдельным цветом** (у нас стены и крыши в одном материале — разделим):
1. Выдели `buildings`, **Tab**, режим граней (клавиша **3**).
2. Навести на скат крыши, **L** — выделит связанные грани крыши (или вручную
   Shift+клик по скатам).
3. Справа Material → **+ New** → назови «roof», цвет тёмно-коричневый → **Assign**.
4. **Tab** назад.

> Хочешь текстуры (кирпич, черепица)? Скачай бесплатные с *ambientCG* или
> *Poly Haven*, перетащи в поле Base Color. Но для концепта хватит и цветов.

---

## 5. Камера и первый рендер

1. `Add → Camera`. Встань в удобный вид (облети мышью), затем
   **Ctrl+Alt+0** — камера «прыгнет» в текущий вид. Точечно двигать: `N` →
   вкладка View → «Camera to View» галочка, крути вид, потом сними галочку.
2. Движок: справа вкладка **Render** (задняя крышка камеры) → **Render Engine:
   Cycles** (фотореализм). Device: GPU если есть, иначе CPU.
   - Включи **Denoise** (внизу, Sampling → Denoise ✓) — убирает шум.
   - Samples: 128 для превью, 512 для финала.
3. **F12** — рендер кадра. Сохранить: `Image → Save As` (в окне рендера).

> Быстрая альтернатива: движок **Eevee** — не так фотореалистично, но рендерит за
> секунды. Хорош, пока учишься кадрировать.

Быстрые улучшения качества: Render → **Ambient Occlusion** ✓ (мягкие тени в
углах), Film → можно оставить. Добавь пару деревьев: `Add → Mesh → Cone` +
`Cylinder`, покрась в зелёный/коричневый, расставь — оживляет.

---

## 6. HD-облёт (анимация камеры по траектории)

1. `Add → Curve → Path` — появится линия. В режиме редактирования (**Tab**)
   растяни её точки вдоль посёлка (по диагонали над домами), приподними по Z.
2. Выдели **камеру**, потом с **Shift** выдели **Path**. `Ctrl+P → Follow Path`
   — камера «сядет» на траекторию.
3. Чтобы камера смотрела на посёлок: выдели камеру → вкладка **Constraints**
   (значок цепи) → **Add → Track To** → Target: `terrain`. Теперь при движении
   она держит посёлок в кадре.
4. Длина полёта: выдели Path → вкладка **Object Data** (кривая) → **Path
   Animation** → Frames = 240 (это 10 сек при 24 fps).
5. Диапазон: снизу таймлайн End = 240. Нажми **Пробел** — предпросмотр облёта.

**Вывод видео:** Render → **Output**:
- Разрешение 1920×1080, Frame Rate 24.
- File Format: **FFmpeg Video**, Encoding → Container **MPEG-4**, Video Codec
  **H.264**.
- Папку вывода задай.
- `Render → Render Animation` (**Ctrl+F12**). Получишь `.mp4` в HD.

> Cycles-анимация считается долго (каждый кадр — рендер). Для первого раза
> ставь Eevee + 240 кадров — минуты вместо часов.

---

## 7. Что помнить про нашу геометрию

- Это **массинг** (объёмы), не детальные дома. Для крупных планов дом лучше
  взять из `house3d.obj` и доработать (окна, крыльцо, гараж-ворота).
- Координаты — **локальные метры**; привязка к МСК-16 — смещение `ox/oy` в
  первой строке `terrain.obj`.
- Рельеф — настоящая поверхность по топосъёмке, дома реально стоят на своих
  высотах.

## 8. Ускоритель: авто-настройка сцены

В папке `../../pipeline/3d/blender_setup.py` — скрипт, который сам импортирует
модель, чинит нормали, ставит солнце, небо, камеру и материалы, рендерит кадр.

**Через интерфейс:** Blender → вкладка **Scripting** сверху → Open → выбери
скрипт → **Run** (▶). Или из терминала (без окна):
```bash
flatpak run org.blender.Blender --background --python pipeline/3d/blender_setup.py \
    -- результат_v2/3D/village.obj /tmp/render.png
```
