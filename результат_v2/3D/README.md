# 3D-модели посёлка

Построены **из реальных данных**: рельеф — по 578 отметкам высот топосъёмки
(перепад 21 м), застройка — по контурам нашего генплана.

## Файлы

| Файл | Что это |
|---|---|
| `terrain.obj` | 3D-рельеф участка (TIN, 19,6 тыс. вершин), координаты локальные в метрах |
| `village.obj` (+`village.mtl`) | Рельеф + объёмная застройка всего посёлка (163 здания) |
| `house3d.obj` | Модель одноквартирного дома 10×10 (по нашим планам этажей) |
| `рельеф_общий_вид.png` | Общий вид рельефа |
| `квартал_вблизи.png` | Квартал крупным планом — дома на склоне |
| `дом_3d.png` | Рендер дома (своя модель, без чужих водяных знаков) |
| `облёт_посёлка.mp4` | **Видео-облёт** посёлка над рельефом |

## Как открыть

- **Компас-3D**: Файл → Открыть → тип `*.obj` (или через импорт). Рельеф лучше
  как поверхность-подложку.
- **SketchUp** (Pro): Файл → Импорт → OBJ. Дальше — материалы, текстуры, тени.
- **Blender** (бесплатно): File → Import → Wavefront (.obj). Здесь делается
  фотореализм: материалы, солнце, трава, деревья, рендер и облёт в HD.

> Координаты в OBJ — локальные (метры), начало вынесено для удобства. Привязка к
> МСК-16: смещение записано в первой строке `terrain.obj` (`ox`, `oy`) — прибавьте
> его, чтобы вернуть глобальные координаты.

## Честно про уровень

Это **концепт-массинг**: объёмы, посадка на рельеф, пропорции — но не фотореализм.
PNG-рендеры и видео сделаны быстрым движком (matplotlib) как превью. Настоящую
красивую картинку (текстуры кирпича, черепицы, окна, освещение, растительность)
получают из этих же OBJ в Blender/SketchUp/Компасе — геометрия уже готова, дело за
материалами и светом. Скрипты-генераторы — в `../../pipeline/3d/`.
